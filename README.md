# SISTCA

Na unidade curricular de Sistemas Computacionais Avançados (SISTCA) da Licenciatura em Engenharia Eletrotécnica e de Computadores,
foi desenvolvido um lab script com o propósito de aprendermos e ensinarmos a conectar uma plataforma de IoT com um simulador open source de eletrónica. 
O principal objetivo deste sistema consiste na medição da temperatura e humidade, através do sensor DHT22 e, para isso, foi desenvolvido um script, em Python, e é possível aceder ao mesmo neste repositório.
É possível aceder, ainda, aos scripts dos exercícios propostos no capítulo 5.
